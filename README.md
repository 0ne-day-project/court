# Debate-based Truth Verifier

## 프로젝트 개요

본 프로젝트는 하나의 주장(Claim)에 대해 단일 AI의 응답이 아닌,
여러 에이전트 간의 토론 과정을 통해 사실 여부를 판단하는 멀티 에이전트 시스템을 구축하는 것을 목표로 한다.

기존의 단일 LLM 기반 응답 방식은
* 편향된 답변
* 근거 부족
* 불확실성 표현 부족
이라는 한계를 가진다.

이를 해결하기 위해 본 시스템은서로 다른 관점을 가진 에이전트 간의 상호작용과 논리적 충돌을 활용하여 보다 신뢰도 높은 판단을 생성한다.

## 노드 구성
* Supporter (주장 옹호자)
  * 입력된 주장을 사실이라고 가정
  * 논리적 근거를 기반으로 주장 강화
  * 상대의 반박을 분석하고 재반박 수행
* Skeptic (회의론자)
  * 입력된 주장을 비판적으로 검토
  * 논리적 오류 및 반례 제시
  * 상대 주장에 대한 공격 수행
* Fact Checker (팩트 검증자)
  * Supporter와 Skeptic의 주장을 종합 분석
  * 객관적인 사실 기준으로 검증 수행
  * 신뢰 가능한 정보와 불확실한 정보를 구분
* Judge (최종 판단자)
  * Fact Checker의 결과를 기반으로 최종 결론 도출
